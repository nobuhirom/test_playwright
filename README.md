# CRM System with Playwright Tests

顧客管理システム（CRM）のプロジェクトです。Next.js、TypeScript、TailwindCSSを使用して構築されています。

## 主な機能

- 顧客管理
- 案件管理
- タスク管理
- ダッシュボード表示
- ユーザー認証

## 技術スタック

- Next.js 14
- TypeScript
- TailwindCSS
- MongoDB
- NextAuth.js
- Playwright (E2Eテスト)

## 開発環境のセットアップ

1. リポジトリのクローン
```bash
git clone https://github.com/nobuhirom/test_playwright.git
cd test_playwright
```

2. 依存パッケージのインストール
```bash
npm install
```

3. 環境変数の設定
`.env`ファイルを作成し、必要な環境変数を設定してください。

4. 開発サーバーの起動
```bash
npm run dev
```

## テストの実行

```bash
npm run test
```
